# 🤖 Makine Öğrenmesi Projelerim (Machine Learning Projects)

Bu depoda, veri analitiği, özellik mühendisliği ve makine öğrenmesi algoritmalarını kullanarak geliştirdiğim uçtan uca projelerim yer almaktadır. 

Aşağıdaki bağlantılardan projelerin kaynak kodlarına, notebook'larına ve detaylı açıklamalarına ulaşabilirsiniz:

---

## 🎯 Projeler

* ### 🎬 [Film Tavsiye Sistemi (Movie Recommendation System)](./movie%20recommendation/)
    Kullanıcıların izledikleri filmlere benzer içerikteki diğer filmleri bulmasını sağlayan, İçerik Tabanlı Filtreleme (Content-Based Filtering) ve NLP teknikleri kullanan sistem. 
    *(Not: Projedeki ~190 MB'lık benzerlik matrisi GitHub sınırını aştığı için Kaggle üzerinde host edilmektedir. Detaylar klasör içerisindeki README'dedir).*

* ### 💳 [Kredi Kartı Dolandırıcılık Tespiti (Fraud Detection)](./Fraud_Detection/)
    Dengesiz veri setleri (imbalanced data) üzerinde SMOTE veya downsampling gibi veri manipülasyonu teknikleri ve makine öğrenmesi algoritmaları kullanılarak geliştirilmiş sınıflandırma projesi.

* ### 🧬 [Meme Kanseri Tespiti (Predict Cancer)](./predict%20cancer/)
    Tıbbi veri setleri üzerinden tümörlerin iyi huylu veya kötü huylu olduğunu tahmin eden, Recall metriği optimizasyonuna ve model çıktılarının görselleştirilmesine odaklanan analitik çalışma.

---

## 🛠️ Kullanılan Teknolojiler
- **Kütüphaneler:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
- **Modelleme & Metotlar:** NLP (Metin Vektörleştirme), Sınıflandırma Algoritmaları, Cosine Similarity
